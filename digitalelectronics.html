Digital Electronics Fundamentals

What is Digital Electronics?

Digital electronics deals with systems that use discrete voltage levels (typically 0V and 5V) to represent binary values (0 and 1). Unlike analog electronics, digital systems are immune to noise and distortion.

Number Systems

  • Binary: Base-2 (0,1) - Fundamental system for digital circuits
  • Decimal: Base-10 (0-9) - Human-readable system
  • Octal: Base-8 (0-7) - Used in some computer systems
  • Hexadecimal: Base-16 (0-9, A-F) - Compact representation of binary

Logic Levels & Voltage

  • TTL Logic: 0V = Logic 0, 5V = Logic 1
  • CMOS Logic: 0V-1/3Vdd = Logic 0, 2/3Vdd-Vdd = Logic 1
  • Noise Margin: Difference between actual and threshold voltages
  • Fan-out: Maximum number of inputs a gate can drive

Basic Gates

  • AND: Output=1 only if ALL inputs=1
  • OR: Output=1 if ANY input=1
  • NOT: Output=inverse of input

Universal Gates

  • NAND: NOT + AND - Can implement any logic function
  • NOR: NOT + OR - Can implement any logic function
  • Any digital circuit can be built using only NAND or only NOR gates

Special Gates

  • XOR: Output=1 if inputs are DIFFERENT
  • XNOR: Output=1 if inputs are SAME
  • Used in parity checking, arithmetic circuits

Boolean Algebra & Simplification

Boolean Laws

  • Commutative: A+B = B+A, A·B = B·A
  • Associative: (A+B)+C = A+(B+C)
  • Distributive: A·(B+C) = A·B + A·C
  • Identity: A+0 = A, A·1 = A
  • Complement: A+A' = 1, A·A' = 0

De Morgan's Theorems

  • (A+B)' = A'·B'
  • (A·B)' = A'+B'
  • Convert AND to OR and vice versa
  • Essential for NAND/NOR implementations

Simplification Methods

  • Boolean Algebra: Using laws and theorems
  • Karnaugh Maps: Visual simplification for 2-6 variables
  • Quine-McCluskey: Tabular method for many variables
  • Logic Minimization Software: Espresso algorithm

Combinational Circuits

Arithmetic Circuits

  • Half Adder: Adds 2 bits (Sum, Carry)
  • Full Adder: Adds 3 bits (including carry-in)
  • Ripple Carry Adder: Chain of full adders
  • Carry Lookahead Adder: Faster parallel addition

Data Selectors

  • Multiplexer (MUX): Selects one of many inputs
  • Demultiplexer (DEMUX): Routes input to one of many outputs
  • 2^n inputs, n select lines, 1 output (MUX)
  • 1 input, n select lines, 2^n outputs (DEMUX)

Code Converters

  • Encoder: 2^n inputs to n outputs
  • Decoder: n inputs to 2^n outputs
  • Priority Encoder: Handles multiple active inputs
  • Seven-Segment Decoder: Drives display panels

Sequential Circuits

Flip-Flops

  • SR Flip-Flop: Set-Reset basic memory
  • JK Flip-Flop: Eliminates indeterminate state
  • D Flip-Flop: Data/Delay - Stores 1 bit
  • T Flip-Flop: Toggle - Changes state on clock

Counters

  • Asynchronous: Ripple counters - Simple but slow
  • Synchronous: All flip-flops clocked together
  • Ring Counter: Circulating single 1
  • Johnson Counter: Twisted ring counter

Registers

  • SISO: Serial In, Serial Out
  • SIPO: Serial In, Parallel Out
  • PISO: Parallel In, Serial Out
  • PIPO: Parallel In, Parallel Out
