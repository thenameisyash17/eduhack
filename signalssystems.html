Signals & Systems Practice Questions

What is the difference between continuous-time and discrete-time signals?

▼

Continuous-Time Signals:

  • Defined for all values of time in a continuous interval
  • Represented as x(t) where t is a continuous variable
  • Examples: Analog audio signals, temperature readings over time
  • Can take any amplitude value within a range

Discrete-Time Signals:

  • Defined only at discrete instants of time
  • Represented as x[n] where n is an integer index
  • Examples: Digital audio samples, monthly sales data
  • Amplitude may be continuous or quantized

Key Relationship: Discrete-time signals are often obtained by sampling continuous-time signals at regular intervals (t = nT).

Explain linearity and time-invariance properties of systems

▼

Linearity: A system is linear if it satisfies both:

1. Additivity: T{x₁(t) + x₂(t)} = T{x₁(t)} + T{x₂(t)}

2. Homogeneity: T{ax(t)} = aT{x(t)}

Where T{•} represents the system transformation, and a is any constant.

Time-Invariance: A system is time-invariant if a time shift in the input causes an identical time shift in the output:

If y(t) = T{x(t)}, then T{x(t - t₀)} = y(t - t₀)

Significance:

  • LTI Systems: Systems that are both linear and time-invariant
  • Can be completely characterized by their impulse response
  • Output can be computed using convolution: y(t) = x(t) * h(t)
  • Analysis is simplified in frequency domain using Fourier/Laplace transforms

What is the significance of Fourier Transform in signal analysis?

▼

The Fourier Transform converts a signal from time domain to frequency domain, revealing its frequency components.

Fourier Transform: X(ω) = ∫-∞∞ x(t) e-jωt dt

Inverse Fourier Transform: x(t) = (1/2π) ∫-∞∞ X(ω) ejωt dω

Significance:

  • Spectral Analysis: Identifies frequency components in a signal
  • Filter Design: Essential for designing frequency-selective filters
  • Modulation/Demodulation: Basis for communication systems
  • Convolution Theorem: Convolution in time domain = Multiplication in frequency domain
  • Parseval's Theorem: Energy conservation between time and frequency domains

Applications: Audio processing, image compression (JPEG), speech recognition, wireless communications, and medical imaging (MRI).

Explain the Sampling Theorem and its importance in digital signal processing

▼

Sampling Theorem (Nyquist-Shannon Theorem):

A continuous-time signal with finite bandwidth (B Hz) can be perfectly reconstructed from its samples if the sampling frequency (fs) is at least twice the maximum frequency component (2B).

Condition: fs ≥ 2B

Where: fs = Sampling frequency, B = Signal bandwidth

Key Concepts:

  • Nyquist Rate: 2B samples/second (minimum sampling rate)
  • Nyquist Frequency: B Hz (maximum representable frequency)
  • Aliasing: Occurs when fs < 2B, causing high frequencies to appear as low frequencies
  • Anti-aliasing Filter: Low-pass filter used before sampling to limit bandwidth to ≤ fs/2

Importance in DSP:

  • Enables analog-to-digital conversion without information loss
  • Foundation for all digital communication and audio systems
  • Essential for CD audio (44.1 kHz sampling for 20 kHz audio)
  • Basis for digital image and video processing

Compare Laplace Transform and Z-Transform

▼

Laplace Transform:

X(s) = ∫0∞ x(t) e-st dt, where s = σ + jω

  • Used for continuous-time signals and systems
  • Transforms differential equations to algebraic equations
  • Region of Convergence (ROC) is crucial for system stability analysis
  • Applications: Control systems, circuit analysis, continuous-time filters

Z-Transform:

X(z) = Σn=-∞∞ x[n] z-n, where z = rejω

  • Used for discrete-time signals and systems
  • Transforms difference equations to algebraic equations
  • ROC determines causality and stability of discrete systems
  • Applications: Digital filter design, discrete-time control systems

Relationship: Z-transform is the discrete-time counterpart of Laplace transform, with z = esT where T is sampling period.

Fourier Transform Connection:

  • Fourier Transform = Laplace Transform evaluated on imaginary axis (s = jω)
  • Discrete-Time Fourier Transform (DTFT) = Z-transform evaluated on unit circle (z = ejω)

What is convolution and how is it used in linear systems?

▼

Convolution is a mathematical operation that computes the output of a linear time-invariant (LTI) system given its input and impulse response.

Continuous-Time: y(t) = ∫-∞∞ x(τ) h(t - τ) dτ = x(t) * h(t)

Discrete-Time: y[n] = Σk=-∞∞ x[k] h[n - k] = x[n] * h[n]

Properties of Convolution:

  • Commutative: x(t) * h(t) = h(t) * x(t)
  • Associative: [x(t) * h₁(t)] * h₂(t) = x(t) * [h₁(t) * h₂(t)]
  • Distributive: x(t) * [h₁(t) + h₂(t)] = x(t) * h₁(t) + x(t) * h₂(t)
  • Identity: x(t) * δ(t) = x(t)

Applications in Linear Systems:

  • System Response: Output = Input convolved with Impulse Response
  • Filtering: Convolution implements filtering operations
  • Signal Processing: Used in smoothing, edge detection, correlation
  • Communication: Channel modeling and equalization

Convolution Theorem: Convolution in time domain equals multiplication in frequency domain, making frequency-domain analysis more efficient.
